Arnold 7.3.2.0

A maintenance release focusing on GPU performance optimizations, enhanced global illumination controls, and critical stability fixes for production environments.

01 Highlights

Core Engine v7.3.2.0-core

Improved GPU Rendering Speed

Refactored GPU core to optimize memory bandwidth. Up to 2x performance increase in high-texture scenes.

v7.3.2.0-core
Lighting v7.3.2.0-light

Global Light Sampling

Intelligent light selection algorithm reducing noise in interior scenes by 30% without increased sample counts.

v7.3.2.0-light

ARN-12401 GPU Core

Fixed crash when using nested instancers with motion blur in GPU mode.

ARN-12455 Shading

Corrected energy loss in OSL closures when using multiple importance sampling.

ARN-12398 USD

Volume scattering now correctly obeys light link sets in USD.
