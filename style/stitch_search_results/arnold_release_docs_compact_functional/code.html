Arnold 7.3.2.0

A maintenance release focusing on GPU performance optimizations, enhanced global illumination controls, and critical stability fixes for production environments.

01 Highlights

Core Engine

Improved GPU Rendering Speed

Refactored GPU core to optimize memory bandwidth. Up to 2x performance increase in high-texture scenes. Optimized CUDA kernel execution for Ada Lovelace and enhanced texture caching for UDIM workflows.

v7.3.2.0-core
Lighting

Global Light Sampling

Intelligent light selection algorithm reducing noise in interior scenes by 30% without increased sample counts.

v7.3.2.0-light
Standards

OCIO 2.3 Support

Full integration with the latest OpenColorIO standards for consistent color management across the VFX pipeline.

v7.3.2.0-color

02 Enhancements

Shaders

Standard Surface Updates

Added 'Thin Film' parameter for iridescent effects. Enhanced 'Coat' layer with multi-scattering BRDF for realistic metallic paints.

ARN-9921
Workflow

USD & Hydra Integration

Native motion blur support for volumes and point clouds in Arnold USD procedural. Hydra render delegate v2 now supports all AOV types and custom LPEs.
